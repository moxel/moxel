# WarpDrive API 

warp-api.js is the javascript driver for warp service.

## Development

First run `npm install` to install all of the required packages.

- The end2end test is inside `./warp-api.spec.js`. To run:
    ```bash
    npm run test:api
    ```

- To check the types, run:
    ```bash
    npm run test:flow
    ```
